# Отчет по лабораторной работе №3

## Тема работы

Изучение системы автоматизации сборки CMake.

## Что было сделано

В этой лабораторной работе я продолжила работу с проектом и сделала сборку
через CMake. Для этого был создан главный файл `CMakeLists.txt`, а также
отдельные файлы `CMakeLists.txt` в папках библиотек и приложений.

По домашнему заданию были оформлены:

1. библиотека `formatter_lib`
2. библиотека `formatter_ex_lib`
3. библиотека `solver_lib`
4. приложение `hello_world_application`
5. приложение `solver_application`

Таким образом проект теперь можно собирать не вручную, а через CMake.

## Что находится в репозитории

1. `TASK.md` - текст задания.
2. `REPORT.md` - отчет.
3. `CMakeLists.txt` - основной файл сборки.
4. Папки с библиотеками и приложениями.

## Вывод

Во время выполнения этой лабораторной работы я поняла, как можно описывать
сборку проекта в CMake и как подключать библиотеки и приложения друг к другу.
